# Project Documentation  
## E-Service Platform for Vehicle Rentals  

---

## 1. Introduction  
This project is a web-based vehicle rental system developed for Bhakti Tours and Travels. It aims to convert the existing offline rental process into a fully digital platform.

---

## 2. Organization Profile  
Bhakti Tours and Travels is a vehicle rental service provider based in Udaipur, offering bikes and scooters for tourists.

---

## 3. Problem Statement  
The existing system is manual, requiring phone or in-person booking, which is time-consuming and inefficient.

---

## 4. Proposed Solution  
A MERN-based web platform that allows:
- Online vehicle browsing  
- Real-time availability  
- Booking and payments  
- Admin management system  

---

## 5. Features  
- User authentication  
- Vehicle search and booking  
- Payment system  
- Admin dashboard  
- Feedback system  

---

## 6. Tech Stack  
- Frontend: React  
- Backend: Node.js, Express  
- Database: MongoDB  

---

## 7. System Modules  
- User Module  
- Admin Module  
- Booking Module  
- Payment Module  

---

## 8. Testing  
System tested for:
- Login functionality  
- Booking flow  
- Payment process  
- Admin operations  

---

## 9. Conclusion  
The system improves efficiency, reduces manual work, and provides a better user experience for vehicle rentals.
